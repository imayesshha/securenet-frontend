## 🎯 Problem Statement
In today's digital world, cybersecurity threats are evolving rapidly. SecureNet provides an intuitive platform for real-time threat detection, secure password validation, malicious URL scanning, and comprehensive security dashboards - empowering users to stay ahead of cyber threats.

## ✨ Features
- 🔐 **Secure Authentication** - JWT-based login/register with protected routes
- 🛡️ **Password Strength Analyzer** - Real-time password security scoring
- 🌐 **URL Threat Scanner** - Malicious link detection and safety reports
- 📊 **Security Dashboard** - Real-time analytics and threat overview
- ✨ **Cyberpunk UI** - Modern, animated interface with custom cursor
- 📱 **Responsive Design** - Perfect on desktop, tablet, mobile
- ⚡ **Fast Performance** - Vite + React 19 optimized builds

## 🚀 Quick Start

### Prerequisites
```bash
Node.js 18+
npm/yarn/pnpm
```

### Installation
```bash
git clone https://github.com/imayesshha/SecureNet.git
cd SecureNet/frontend
npm install
npm run dev
```

### Build & Deploy
```bash
npm run build
npm run preview  # Local preview
vercel --prod    # Deploy to Vercel
```

## 🔌 API Endpoints
| Method | Endpoint | Description | Auth |
|--------|----------|-------------|------|
| POST | `/api/auth/register` | User registration | No |
| POST | `/api/auth/login` | User login | No |
| POST | `/api/security/password-check` | Password strength analysis | Yes |
| POST | `/api/security/url-check` | URL threat scan | Yes |
| GET | `/api/user/dashboard` | Security dashboard data | Yes |

## 🛡️ Security Features
- JWT Token Authentication & Authorization
- Protected Routes & Role-based Access
- Input Sanitization & Validation
- Secure Password Hashing (bcrypt)
- CORS Protection
- Rate Limiting
- Helmet Security Headers
